# ODISSEI-Five-Safes-license

This repository hosts the machine-readable version of the ODISSEI Five Safes license, as well as the machine-readable version of the Trusted Research Environment (TRE) vocabulary.

The ODISSEI Five Safes License is developed and maintained by the Dutch national data infrastructure for the social sciences, [ODISSEI](https://odissei-data.nl). [![DOI](https://zenodo.org/badge/1147917256.svg)](https://doi.org/10.5281/zenodo.18456774).

The ODISSEI Five Safes License defines a standardised set of conditions for accessing and using sensitive datasets in a Trusted Research Environment (TRE), according to the Five Safes principles.

The Trusted Research Environment vocabulary defines ODRL actions and RDF classes that are used in the Five Safes License. [![URI](https://img.shields.io/badge/URI-to_be_defined-blue)](https://w3id.org/).
